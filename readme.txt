== Description ==

A WooCommerce plugin that apply CSS3 Tags & Stitched Effect style design for WooCommerce product meta (SKU, categories and tags)

In WooCommerce Sidebar Panel, there will be a new menu link called 'Product CSS3 Tags' where you can update the options.

The list of options for the section above:

*   Enabled Product CSS3 Design
*   Show Product SKU  
*   Show Product Categories
*   Show Product Tags
*   Use CSS3 Stitched Effect that apply for SKU and Categories
*   Product Tags Color Scheme
*   Change SKU text to any text you prefer (for example: Model, Product Code)

== Installation ==

1. Upload the entire *woocommerce-product-css-tags* folder to the */wp-content/plugins/* directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to WooCommerce panel at left sidebar menu and update the options there.
4. That's it. You're ready to go and cheers!
